# ✨ Daily Inspiration Hub

A beautiful, responsive web application that delivers motivational quotes and daily inspiration at your fingertips.

![Quote App Screenshot](https://via.placeholder.com/800x400/6366f1/ffffff?text=Daily+Inspiration+Hub)

## 🌟 Features

- **100+ Curated Quotes** — Hand-picked motivational, success, life, wisdom, and creativity quotes
- **Category Filtering** — Browse quotes by category: Motivation, Success, Life, Wisdom, Creativity
- **Favorites System** — Save your favorite quotes (persisted in localStorage)
- **Copy to Clipboard** — Share quotes with one click
- **Keyboard Shortcuts** — Space (new quote), C (copy), F (favorite)
- **Beautiful UI** — Smooth animations, gradient backgrounds, glassmorphism design
- **Fully Responsive** — Works perfectly on desktop, tablet, and mobile
- **Zero Dependencies** — Pure HTML, CSS, and JavaScript
